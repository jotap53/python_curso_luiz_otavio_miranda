O curso mais completo de Python 3 disponível atualmente, com **mais de 140 horas de conteúdo** e **745+ aulas**, cobrindo desde lógica de programação até padrões de projeto avançados. Constantemente atualizado para as versões mais recentes da linguagem.

## ⚙️ Como Executar

**Pré-requisitos:** Python 3.x instalado ([download](https://www.python.org/downloads/))

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/curso-python.git

# Acesse o diretório
cd curso-python

# (Opcional) Crie um ambiente virtual
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Instale dependências (quando necessário)
pip install -r requirements.txt
```
